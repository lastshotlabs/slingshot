The app root config controls everything your app needs to run. Here is the full surface.

## A production-ready app root

```typescript title="src/index.ts"
import { createServer } from '@lastshotlabs/slingshot';
import { createAuthPlugin } from '@lastshotlabs/slingshot-auth';
import { createPermissionsPlugin } from '@lastshotlabs/slingshot-permissions';
import { blogPackage } from './packages/blog';
import { adminPackage } from './packages/admin';

await createServer({
  // --- Server ---
  port: Number(process.env.PORT ?? 3000),

  // --- App metadata ---
  meta: { name: 'My App', version: '1.0.0' },

  // --- Security ---
  security: {
    signing: { secret: process.env.JWT_SECRET! },
    cors: { origin: ['https://myapp.com', 'https://admin.myapp.com'] },
    csrf: { enabled: true },
    rateLimit: { windowMs: 60_000, max: 100 },
    helmet: { enabled: true },
  },

  // --- Database ---
  db: {
    default: { adapter: 'postgres', url: process.env.DATABASE_URL! },
    cache: { adapter: 'redis', url: process.env.REDIS_URL! },
  },

  // --- Plugins ---
  plugins: [
    createAuthPlugin({
      auth: { roles: ['user', 'editor', 'admin'], defaultRole: 'user' },
    }),
    createPermissionsPlugin(),
  ],

  // --- Packages ---
  packages: [blogPackage, adminPackage],

  // --- Events ---
  eventBus: 'bullmq', // or 'memory' (default), 'redis'

  // --- WebSockets ---
  ws: {
    endpoints: {
      '/ws': {
        onConnect: async (ws) => { console.log('Connected:', ws.data.actorId); },
        onMessage: async (ws, message) => { /* handle message */ },
        onClose: async (ws) => { console.log('Disconnected'); },
      },
    },
  },

  // --- SSE ---
  sse: {
    endpoints: {
      '/events': { heartbeatMs: 30_000 },
    },
  },

  // --- Jobs ---
  jobs: { enabled: true },

  // --- Operations ---
  logging: { enabled: true },
  metrics: { enabled: true },
  observability: { serviceName: 'my-app' },

  // --- Secrets ---
  secrets: { provider: 'env' },
});
```

## Config reference

### Server

| Key    | Type     | Default | What it does      |
| ------ | -------- | ------- | ----------------- |
| `port` | `number` | `3000`  | HTTP server port  |

### App metadata

| Key             | What it does                                         |
| --------------- | ---------------------------------------------------- |
| `meta`          | App name and version for OpenAPI docs                |
| `routesDir`     | Directory for auto-discovered route files            |
| `modelSchemas`  | Directory for shared Zod schemas preloaded at boot   |
| `versioning`    | API versioning config for versioned route trees      |

### Security

```typescript
security: {
  signing: {
    secret: 'min-32-char-secret',
    previousSecrets: ['old-secret'],    // for zero-downtime rotation
  },
  cors: {
    origin: ['https://myapp.com'],      // allowed origins
    credentials: true,
  },
  csrf: { enabled: true },
  rateLimit: {
    windowMs: 60_000,                   // 1 minute window
    max: 100,                           // max requests per window
  },
  helmet: { enabled: true },           // security headers
}
```

### Database

```typescript
// Single database
db: {
  default: { adapter: 'postgres', url: process.env.DATABASE_URL! },
}

// Multiple named databases
db: {
  default: { adapter: 'postgres', url: process.env.DATABASE_URL! },
  cache: { adapter: 'redis', url: process.env.REDIS_URL! },
  analytics: { adapter: 'mongo', url: process.env.MONGO_URL! },
}

// SQLite for development
db: {
  default: { adapter: 'sqlite', url: './data/app.db' },
}
```

Supported adapters: `postgres`, `sqlite`, `mongo`, `redis`. Entities use the `default`
connection unless they specify a named connection with `adapter: 'cache'`.

### Composition

| Key           | What it does                                          |
| ------------- | ----------------------------------------------------- |
| `packages`    | Feature packages to load (the main composition surface) |
| `plugins`     | Raw plugins for auth, permissions, and infrastructure |
| `permissions` | Shared permissions bootstrap config                   |

### Events

| Key               | What it does                                        |
| ----------------- | --------------------------------------------------- |
| `eventBus`        | Event transport: `'memory'` (default), `'bullmq'`, `'redis'` |
| `kafkaConnectors` | Kafka bridge config for external event streaming    |

### Realtime

```typescript
// WebSocket endpoints
ws: {
  endpoints: {
    '/ws': {
      onConnect: async (ws) => { },
      onMessage: async (ws, message) => { },
      onClose: async (ws) => { },
    },
  },
}

// SSE endpoints
sse: {
  endpoints: {
    '/events': { heartbeatMs: 30_000 },
  },
}
```

### Infrastructure

| Key             | What it does                                       |
| --------------- | -------------------------------------------------- |
| `upload`        | File upload config (provider, directory, limits)   |
| `jobs`          | Background job orchestration config                |
| `runtime`       | Runtime abstraction (Bun, Node, edge, Lambda)      |
| `secrets`       | Secret provider: `'env'`, `'ssm'`, `'file'`       |

### Operations

| Key             | What it does                                       |
| --------------- | -------------------------------------------------- |
| `logging`       | Structured logging config                          |
| `metrics`       | Metrics endpoint (`/metrics`)                      |
| `observability` | OpenTelemetry tracing config                       |
| `tenancy`       | Tenant resolver for multi-tenant apps              |
| `middleware`     | Global middleware array                            |
| `validation`    | Request/response validation behavior               |

## Config by use case

### Minimal dev server

```typescript
await createServer({
  port: 3000,
  packages: [appPackage],
});
```

### API with auth

```typescript
await createServer({
  port: 3000,
  security: { signing: { secret: 'dev-secret-change-in-prod' } },
  plugins: [createAuthPlugin({ auth: { roles: ['user'], defaultRole: 'user' } })],
  packages: [appPackage],
});
```

### Multi-tenant SaaS

```typescript
await createServer({
  port: 3000,
  security: { signing: { secret: process.env.JWT_SECRET! } },
  db: { default: { adapter: 'postgres', url: process.env.DATABASE_URL! } },
  tenancy: {
    resolver: 'header',
    headerName: 'x-tenant-id',
  },
  plugins: [
    createAuthPlugin({ auth: { roles: ['user', 'admin'], defaultRole: 'user' } }),
    createPermissionsPlugin(),
  ],
  packages: [appPackage],
});
```

### Realtime app with WebSockets

```typescript
await createServer({
  port: 3000,
  security: { signing: { secret: process.env.JWT_SECRET! } },
  plugins: [createAuthPlugin({ auth: { roles: ['user'], defaultRole: 'user' } })],
  ws: {
    endpoints: {
      '/ws': {
        onConnect: async (ws) => {
          console.log('User connected:', ws.data.actorId);
        },
        onMessage: async (ws, message) => {
          // Echo back
          ws.send(JSON.stringify({ echo: message }));
        },
      },
    },
  },
  packages: [chatPackage],
});
```
